## üöÄ Getting Started

### Quick Start (30 minutes)
1. **Read** [`QUICK_START.md`](./QUICK_START.md)
2. **Add** documents to `personal-files/`
3. **Run** Profile Analyzer prompt
4. **Begin** applying with confidence
